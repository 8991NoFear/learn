Chart JS

Usage

  1. Cách viết code tiêu biểu

    
    const ctx = document.getElementById('mChart');
    const labels = ['Jan', 'Feb', 'Mar'];
    const data = [10, 20, 30];
    const config = {
        type: 'line',
        data: {
            labels: labels,
            datasets: [{
                label: 'name of dataset'
                data: data,
                backgroundColor: 'blue',
                borderColor: 'blue',
            }]
        },
        config: {
            scales: {
                y: { // defining min and max so hiding the dataset does not change scale range
                    min: 0,
                    max: 100
                }
            },
            aspectRatio: 3 // width/height ratio
        },
        plugins: []
    }
    new Chart(ctx, cfg);

  2. Mặc định ChartJS là responsive, nghĩa là width và height của thẻ canvas chỉ nhằm mục đính giữ chỗ.

Others

  1. Thuộc tính datasets của data phải là một mảng các object

  2. Thuộc tính data của datasets thường là một mảng số liệu. Tuy nhiên nó có thể nhận một mảng object, khi đó cần chỉ định thêm thuộc tính parsing

  3. Muốn hiển thị số trong cột của 'bar chart' thì sử dụng thêm plugins: chartjs-plugin-datalabels

  4. Configure options trong thuộc tính options sẽ áp dụng cho toàn bộ chart. Ngược lại configure options trong từng dataset của datasets thì chỉ áp dụng cho mỗi dataset đó thôi
