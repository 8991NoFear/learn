1) Vue CLI
$ npm install -g @vue/cli
$ vue --version
$ vue create hello-vue
$ cd hello-vue
$ npm run serve
$ npm run build

2) Cấu trúc thư mục
- src (source codel)
    |- main.js (file gốc)
    |- App.vue (component gốc)
    |- assets (images, fonts, ...)
    |- components (chứa component con)

3) Component
Mỗi component đều gồm có 3 thành phần:
+, template: html
+, script: js
+, style: css
    +, có hỗ trợ scss
    +, thẻ <style scoped>: style dành riêng cho component

Tạo component
+, Tạo file .vue
+, Import
+, Khai báo
+, Sử dụng

Mỗi một component chỉ có 1 template, mỗi một template chỉ có 1 element (thông thường là <div>)

2 ways binding:
+, ràng buộc 2 chiều
+, phụ thuộc vào loại element

hook: hàm móc vào những thời điểm nhất định trong life cycle

binding:
+, value binding: {{  }}
+, html attribute binding: v-bind:[attr]="" or :[attr]=""
+, class biding: (:class="{red: isRed, blue: isBlue}")

Model: liên kết element với property (v-model)

Event Handling: bắt sự kiện của element
+, v-on:[name_of_event]="js code goes here" or @[name_of_event]="js code goes here"
+, prevent default event: e.g. @click.prevent="abc()"

ref: ánh xạ đến chính element
+, ref="any" -> truy cập vào element this.$refs.any

Conditional rendering:
    +, v-show, v-hide (can thiệp vào display)
    +, v-if, v-else-if, v-else

List rendering:
    +, v-for

Methods:
    +, hàm của vue object

Watch:
    +, theo dõi sự thay đổi của thuộc tính của vue object
    +, tên giống hệt tên thuộc tính
